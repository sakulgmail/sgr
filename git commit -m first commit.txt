For production, you should switch to your real app URL (domain or fixed host), for example:

https://gsr.yourcompany.com (recommended), or
http://10.x.x.x:5173 if staying internal-only
Then update env values accordingly:

APP_BASE_URL=<production-web-url>
VITE_API_BASE_URL=<production-api-url>
So you don’t keep changing per machine, use a stable DNS name in production instead of raw IP.
